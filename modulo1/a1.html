tudo do modulo 1

estilos de letras

negrito = strong italico = em sublinhado = ins deletado = del marca texto = mark (podemos pintar com backgrond-color ou so as letras com color)

criando listas

listas ordenadas = ol (temos tipos: 1, A, a, I) listas nao ordenadas = ul (existes tipos) itens na lista = li lista de definição = dl termo = dt definição = dd

criar tabelas

para criar tabelas se usa a tag TABLE, dentro dela usamos TR para linhas e TH para cabeçalhos e TD para celulas

nome idade cidade
maria 19 anos rio de janeiro
joao 20 anos sao paulo

para cada parte da tabela precisa abrir um TR e por as informaçoes

tipos de textos

texto subscrito = sub texto sobescrito = sup citacoes simples = q citacoes complexas = blockquote abreviacoes = abbr texto ao contrario = bdo:l ou bdo:r

imagens

para adicionar imagens usamos IMG
podemos copiar o endereço de uma foto no google e usa-la
para uma imagem se adaptar ao tamanho de tela se usa a tag PICTURE, assim ela abrira a menor imagem na menor tela e a maior imagem na maior tela
usamos source P source M img padrao tamanho G

tamanho das imagens

opte sempre por jpg ou png tamanho para celular: 300px tamanho para tablet: 700px tamanho paca tv ou computador: 1000px e importante nao deixar muito grande e nem com qualidade muito alta para nao pesar o site

adicionamos simbolos e emojis

® = marca registrada © = copyrigth ™ = trade mark € = euro £ = libra ¥ = yen ¢ = centavo ∅ = vazio ∑ = soma Δ = Delta ← = seta esquerda ↑ = seta acima → = seta direita ↓ = seta abaixo

para adicionar emojis usamos

&#x + codigo do emoji que voce queira adicionar (emojipedia ou codepoints)
